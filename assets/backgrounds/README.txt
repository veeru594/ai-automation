# Static Backgrounds Required

This directory must contain the following files:

- bg_cover.jpg    (1080x1350) - Cover slide background
- bg_news.jpg     (1080x1350) - News slide background  
- bg_insight.jpg  (1080x1350) - Market insight slide background
- bg_cta.jpg      (1080x1350) - CTA slide background

If any file is missing, the generation script will fail with a clear error message.
